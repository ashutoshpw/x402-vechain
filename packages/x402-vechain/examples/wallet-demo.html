1. Detect Available Wallets

Check which VeChain wallets are installed in your browser.

2. Connect to Wallet

Connect to an available VeChain wallet (VeWorld or Connex).

3. Make Payment-Protected Request

Fetch premium content that requires payment. The wallet will automatically sign the payment.

Usage Example

import { x402Fetch, autoDetectWallet } from '@x402/vechain';

// Auto-detect wallet
const wallet = autoDetectWallet();

// Make request with automatic payment
const response = await x402Fetch('https://api.example.com/premium', {
  facilitatorUrl: 'https://facilitator.example.com',
  wallet,
  maxAmount: '1000000000000000000', // Max 1 VET
});

const data = await response.json();
