## What is x402.vet?

x402.vet is a payment facilitator implementing the [x402 protocol](https://github.com/coinbase/x402) for VeChain blockchain. It enables seamless cryptocurrency payments using VeChain's native tokens (VET, VTHO, VEUSD, B3TR) with optional fee delegation for gasless transactions.

## Key Features

- ✅ **x402 Protocol Compliant** - Standard payment protocol
- ✅ **VeChain Native** - Built specifically for VeChain blockchain  
- ✅ **Wallet Integration** - Support for VeWorld, Connex (Sync/Sync2)
- ✅ **Fee Delegation** - Optional gas sponsorship for users
- ✅ **TypeScript First** - Full type safety and IntelliSense
- ✅ **Production Ready** - Rate limiting, CORS, error handling
